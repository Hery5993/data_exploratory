# 📊 Data Exploratory Analysis with Pandas

This repository contains Jupyter notebooks and Python scripts for performing **Exploratory Data Analysis (EDA)** using the **Pandas** library.  
The goal is to demonstrate best practices in cleaning, transforming, analyzing, and visualizing datasets before applying advanced analytics or machine learning models.

---

## 🚀 Features
- Data loading from CSV, Excel, SQL, and APIs
- Data cleaning (handling missing values, duplicates, outliers)
- Data transformation (grouping, merging, filtering, pivoting)
- Descriptive statistics and summary measures
- Exploratory visualization with:
  - Pandas built-in plotting
  - Matplotlib & Seaborn (optional)
- Example workflows for real-world datasets
